# Expense Tracker Backend

A backend service built using Node.js, Express.js, and MongoDB to manage user expenses through RESTful APIs.

## Features
- Create, view, update, and delete expenses
- RESTful API design with clean routing
- Modular architecture using routes, controllers, and models
- MongoDB integration using Mongoose

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose

## Project Structure
- routes/ – API endpoints
- controllers/ – Business logic
- models/ – Database schemas

## Usage
This project was used as a backend learning reference to understand real-world API design, data handling, and database interaction.
